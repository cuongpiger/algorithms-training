Think of it as solving a quadratic equation for \(k\), then taking the integer part.

## From inequality to quadratic

You need the largest integer \(k\) such that
\[
\frac{k(k+1)}{2} \le n
\]
Multiply both sides by 2:
\[
k(k+1) \le 2n \Rightarrow k^2 + k - 2n \le 0
\]
So you look at the quadratic equation
\[
k^2 + k - 2n = 0
\]
and solve it with the quadratic formula.

For a quadratic \(ak^2 + bk + c = 0\),
\[
k = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
\]
Here \(a=1, b=1, c=-2n\), so
\[
k = \frac{-1 \pm \sqrt{1 - 4\cdot 1\cdot(-2n)}}{2}
= \frac{-1 \pm \sqrt{1 + 8n}}{2}
\]
You ignore the negative root (because \(k\ge 0\)), so
\[
k = \frac{-1 + \sqrt{1 + 8n}}{2}-
\]

## Where \(\sqrt{2n+0.25} - 0.5\) comes from

Observe:
\[
\frac{-1 + \sqrt{1 + 8n}}{2}
= -\frac{1}{2} + \frac{1}{2}\sqrt{1 + 8n}
\]
Factor 4 inside the square root:
\[
\frac{1}{2}\sqrt{1 + 8n}
= \frac{1}{2}\sqrt{4\left(2n + 0.25\right)}
= \frac{1}{2}\cdot 2\sqrt{2n + 0.25}
= \sqrt{2n + 0.25}
\]
So
\[
k = -\frac{1}{2} + \sqrt{2n + 0.25}
= -0.5 + \sqrt{2n + 0.25}
\]

Because you need an integer number of rows and the true answer is “largest \(k\) with the inequality,” you take the floor:
\[
k = \left\lfloor -0.5 + \sqrt{2n + 0.25} \right\rfloor
\]
